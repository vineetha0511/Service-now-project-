# Virtual Agent-Driven SLA Breach Awareness & Justification System

## Project Overview
This project was developed on the ServiceNow platform to proactively monitor Incident SLAs, identify potential SLA breach risks, automate email notifications, enforce SLA governance through UI and Data Policies, and provide reporting for SLA compliance. The solution improves accountability and helps organizations maintain audit-ready SLA processes.

## Features
- Incident SLA Monitoring
- SLA Breach Awareness
- Automated Email Notifications using Flow Designer
- Custom SLA Breach Reason & Justification Fields
- UI Policies and Data Policies
- SLA Reports and Dashboard
- SLA Governance and Accountability
